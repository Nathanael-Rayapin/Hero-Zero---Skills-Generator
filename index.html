Hero Zero

1 - Que ce soit dans votre inventaire ou dans votre coffret bancaire, indiquez le nombre d'objets que vous voulez comparer, puis confirmez.

2 - Pour chaque(s) Item(s) crée(s), indiquez le type de l'item ainsi que leurs caractéristiques (0 si l'item ne possède pas une caractéristique)

3 - Une fois tous les champs complétés, cliquez sur le button Générateur

** Le Générateur va vous renvoyer les items qui avantageront votre personnage (en vert). Vous pourrez aussi voir le total que ces objets vous donnent dans chaque caractéristique **
